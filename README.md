# CryptoPulse: Your All-in-One Crypto Companion

**CryptoPulse** is a powerful and intuitive mobile application for cryptocurrency enthusiasts, built with React Native and Expo. It provides a comprehensive suite of tools to track, analyze, and manage your digital asset portfolio. Leveraging the power of Google's Gemini AI, CryptoPulse offers unique insights and analysis to help you make informed decisions in the fast-paced world of crypto.

- **Real-time Market Overview**: Stay updated with the latest prices, market caps, and 24-hour changes for top cryptocurrencies.
- **Detailed Coin Analysis**: Dive deep into specific cryptocurrencies with historical price charts, market stats, and detailed information.
- **AI-Powered Price Analysis**: Utilize the integrated Gemini AI to get sophisticated price analysis and predictions for any cryptocurrency.
- **Portfolio Tracking**: Monitor your investment performance with a clear summary of your total portfolio value and asset allocation.
- **Interactive Charts**: Visualize your portfolio distribution with interactive pie charts and analyze historical performance with bar charts.
- **Customizable Watchlist**: Keep a close eye on your favorite cryptocurrencies with a personalized watchlist.
- **Sleek & Modern UI**: A clean, user-friendly interface with both light and dark modes for a comfortable viewing experience.
- **Cross-Platform**: Built with Expo, CryptoPulse is ready to be deployed on both Android and iOS devices.

## üöÄ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (LTS version recommended)
- [Expo Go](https://expo.dev/client) app on your Android or iOS device for testing.
- [Android Studio](https://developer.android.com/studio) or [Xcode](https://developer.apple.com/xcode/) for running on emulators/simulators.

### Installation & Setup

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/your-username/cryptopulse.git
    cd cryptopulse
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    ```

3.  **Set up your environment variables:**

    Create a `.env` file in the root of the project and add your Gemini API key:

    ```env
    VITE_GEMINI_API_KEY=your_gemini_api_key
    ```

4.  **Run the development server:**

    ```bash
    npm start
    ```

    This will start the Metro bundler.

5.  **Run on your device or emulator:**

    - **On your physical device**: Scan the QR code from the Metro bundler terminal with the Expo Go app.
    - **On an Android Emulator**: Press `a` in the terminal.
    - **On an iOS Simulator**: Press `i` in the terminal.
